Тест hover-эффекта для каталога

Десктопная версия

  • Кабинет руководителя
  • Мебель для персонала
  • Столы переговоров
  • Стулья и кресла
О НАС ДИЗАЙН-ПРОЕКТ СЕРВИС КОНТАКТЫ

Инструкция для тестирования:

  1. 1. Наведите курсор на кнопку "КАТАЛОГ"
  2. 2. Должно появиться выпадающее меню с плавной анимацией
  3. 3. Уберите курсор - меню должно плавно исчезнуть
  4. 4. При наведении на пункты меню они должны подсвечиваться

Мобильная версия (остается с кликом)

  • Кабинет руководителя
  • Мебель для персонала
  • Столы переговоров
  • Стулья и кресла

Инструкция для мобильной версии:

  1. 1. Нажмите на кнопку "КАТАЛОГ"
  2. 2. Меню должно появиться/скрыться
  3. 3. Нажмите еще раз, чтобы скрыть
