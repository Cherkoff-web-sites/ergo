Инструкция для тестирования:

  1. 1. Запустите React-приложение: npm run dev
  2. 2. Перейдите на любую страницу каталога

  4. 4. Должно появиться выпадающее меню с плавной анимацией
  5. 5. Уберите курсор - меню должно плавно исчезнуть
